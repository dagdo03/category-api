List Endpoint

Category:  

1. GET /api/category:  
   Untuk mendapatkan seluruh kategori yang ada.
2. POST /api/category:  
   Untuk membuat kategori baru.
3. DELETE /api/category/{id}:  
   Untuk menghapus kategori dengan id tertentu.
4. PUT /api/category/{id}:  
   Untuk Mengupdate data kategori dengan id tertentu.
5. GET /api/category/{id}:  
   Untuk mendapatkan data kategori dengan ID tertentu.

Product:  
1. GET /api/product:  
   Untuk mendapatkan seluruh produk yang ada.
2. POST /api/product:  
   Untuk membuat produk baru.
3. DELETE /api/product/{id}:  
   Untuk menghapus produk dengan id tertentu.
4. PUT /api/product/{id}:  
   Untuk Mengupdate data produk dengan id tertentu.
5. GET /api/product/{id}:  
   Untuk mendapatkan data produk dengan ID tertentu.
